About this project

This project follows the MVC (Model-View-Controller) architectural pattern using Express.js as the backend framework.

The Model layer handles all interactions with a MySQL database, such as user data storage and retrieval. The MySQL database runs inside a Docker container, providing an isolated and consistent environment for development and testing.

The Controller layer contains the business logic and routes that connect the client with the database, responding to user actions and API requests.

The View layer, although minimal in this setup, is used for rendering HTML pages or sending JSON responses.

To manage authentication securely, the project uses JWT (JSON Web Tokens), allowing users to log in and receive a token that must be included in future requests to access protected routes.

This ensures that only authorized users—such as admins—can perform sensitive operations like creating, updating, or deleting user records.
